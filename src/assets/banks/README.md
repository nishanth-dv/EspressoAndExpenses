# Bank logos (optional drop-in)

`BankLogo` renders a brand-colored monogram by default. To use a real logo for a
bank, drop an SVG here named by the bank's slug — lowercase, non-alphanumerics
replaced with `-`. `BankLogo` picks it up automatically (no code change).

Slugs for the built-in `BANKS` list:

| Bank | File |
|---|---|
| HDFC | `hdfc.svg` |
| ICICI | `icici.svg` |
| SBI | `sbi.svg` |
| Axis | `axis.svg` |
| Kotak | `kotak.svg` |
| IDFC | `idfc.svg` |
| Yes Bank | `yes-bank.svg` |
| IndusInd | `indusind.svg` |
| Amex | `amex.svg` |
| Citi | `citi.svg` |
| Standard Chartered | `standard-chartered.svg` |
| RBL | `rbl.svg` |

Source logos only from each bank's official brand/press kit or a freely licensed
source (e.g. Wikimedia Commons). Square-ish, transparent-background SVGs render
best.
